Catálogo de Filmes - MVC PHP
Descrição

Sistema desenvolvido em PHP utilizando o padrão MVC (Model-View-Controller).

O sistema permite visualizar um catálogo de filmes armazenados em banco de dados MySQL.

Tecnologias
PHP
MySQL
PDO
Bootstrap
HTML
Estrutura MVC
Config: conexão com banco de dados.
Models: acesso aos dados.
Controllers: controle da lógica.
Views: interface do usuário.
Banco de Dados

Banco utilizado:

catalogo_filmes

Tabela principal:

filmes

Campos:

id
titulo
genero
ano
sinopse
data_criacao
Como Executar
Importar o arquivo catalogo_filmes.sql no phpMyAdmin.
Colocar o projeto na pasta htdocs do XAMPP.
Iniciar Apache e MySQL.
Acessar:

http://localhost/CATALAGO_FILMES2
